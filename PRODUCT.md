# PRODUCT.md — O.T Performance

register: brand

## What this is
A bilingual (Hebrew-first RTL / English) capability-profile marketing site for O.T Performance,
a performance-marketing, data and automation agency led by Tamir & Ofir.
Static site: index.html + pages/ (about, privacy, terms, accessibility). No backend.

## Users
Israeli SMB owners and marketing managers evaluating an agency. They arrive from
WhatsApp referrals or paid/organic traffic. Primary action: send a WhatsApp message
or leave details in the lead form. Secondary: read the capability document and build trust.

## Brand voice
Precise, engineered, honest. "We measure, we don't guess." No hype numbers,
no invented stats, no fake logos or testimonials. Trust is built through the
document metaphor itself: a confidential capability profile, not a salesy landing page.

## Approved design identity (DO NOT REPLACE)
Luxury printed-document concept: warm off-white paper (#F7F5F0) floating on a deep
dark surround, electric cobalt accent (#2F5BF0), numbered sections 00–09, bilingual
labels (mono English microtype over Hebrew titles). Improvements must refine this
identity, never swap it for a generic SaaS look.

## Anti-references
- Generic dark-gradient SaaS templates
- Fake metrics / invented social proof (hard ban, user rule)
- Emojis anywhere in the UI (hard ban, user rule) — SVG icons only

## Conversion model
WhatsApp is the primary channel (button in hero, contact, floating). The form has no
backend: it must bridge into WhatsApp with a prefilled message. Phone number is
currently a placeholder (972500000000) pending the real number from the client.
